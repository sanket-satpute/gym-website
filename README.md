# Iron Forge Fitness

> **Transform Your Body. Transform Your Life.**

Premium conversion-focused fitness brand website for Iron Forge Fitness, Chandrapur.

## Tech Stack

- **HTML5** — Semantic, SEO-optimized single-page structure
- **CSS3** — Custom properties, Grid, Flexbox, scroll animations
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Google Fonts** — Poppins (headings) + Inter (body)
- **Lucide Icons** — Lightweight icon library via CDN
- **Unsplash** — High-quality stock imagery

## Project Structure

```
Gym-Website/
├── index.html              # Single-page HTML (20+ sections)
├── css/
│   ├── variables.css       # Design tokens & custom properties
│   ├── base.css            # CSS reset & utility classes
│   ├── layout.css          # Container & grid systems
│   ├── components.css      # Reusable UI components
│   └── sections.css        # Section-specific styles
├── js/
│   ├── main.js             # Interactions & animations
│   └── form.js             # Form validation
├── assets/
│   ├── images/             # Image assets
│   └── videos/             # Video assets
└── README.md
```

## Design System

| Token | Value | Usage |
|-------|-------|-------|
| Primary (60%) | `#FFFFFF` | Backgrounds, breathing space |
| Secondary (30%) | `#0F172A` | Hero, dark sections, headings |
| Accent (10%) | `#EF4444` | CTAs, buttons, highlights |
| Body Text | `#64748B` | Paragraphs, descriptions |
| Borders | `#E2E8F0` | Cards, inputs, dividers |

## Getting Started

Simply open `index.html` in a browser. No build step required.

```bash
# Or use a local server
npx serve .
```
